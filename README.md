# GIROPie × C-Square Collection Workflow Presentation

This repository is a presentation-style React app (Vite + Tailwind + Framer Motion) demonstrating the end-to-end eMandate-driven collection workflow between Seller ERP, Buyer, Platform, Easebuzz, NPCI, and ERP sync.

Getting started

1. Install dependencies:

```bash
npm install
```

2. Run dev server:

```bash
npm run dev
```
